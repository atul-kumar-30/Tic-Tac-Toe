This is a classic Tic-Tac-Toe game built as a simple web application. It allows two players to play against each other on a single screen, with a clear indication of the current player and the game's outcome.

Features
Two-Player Gameplay: Take turns with a friend to place your 'X' or 'O' on the board.

Win/Draw Detection: The game automatically detects and announces the winner or a draw.

Game Status Display: A message at the top of the board shows whose turn it is or the final result of the game.

Restart Button: Easily start a new game at any time with the "Restart Game" button.

Technologies Used
HTML: Structures the game board, including the cells, status display, and restart button.

CSS: Styles the grid layout, colors, and fonts to create a clean and intuitive game interface.

JavaScript: Manages the entire game logic, including player turns, marking cells, and checking for winning combinations.

How to Run
Save the files: Make sure you have the index.html, style.css, and script.js files saved in the same folder.

Open in a Browser: Simply open the index.html file in your preferred web browser (like Chrome, Firefox, or Edge).

Start Playing: Click on any empty cell to place your marker and start the game!
